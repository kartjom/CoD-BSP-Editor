# Call of Duty BSP Editor

**A powerful tool for Call of Duty 1/UO modders to easily edit compiled maps.**

## Introduction

The Call of Duty BSP Editor is a user-friendly tool designed to make editing compiled maps easier than ever before. Modders and map creators can now perform complex changes and modifications in minutes. This tool removes the previously challenging barriers to map editing, allowing you to unleash your creativity without the need of decompiling the map.

## Features

With the BSP Editor, you can:

- **Edit Entities**: Remove, edit, or add any entity to the map.
- **Material Replacement**: Replace materials used by the map to give it a fresh look.
- **Brush Modification**: Add or modify existing brushes for custom collisions.
- **Vertex Editing**: Edit any vertex position on the map for fine-tuned adjustments. Best used with my [other tool](https://github.com/kartjom/bsp-converter).
- **Gamemode Integration**: Easily incorporate multiplayer gamemodes such as Capture The Flag and Search and Destroy.

## Prerequisites

Before you get started with the editor, ensure that you have following dependencies installed:

- [.NET 6 Runtime](https://dotnet.microsoft.com/download/dotnet/6.0) - You'll need the .NET 6 runtime to run this tool.

Now you're all set, happy modding!
